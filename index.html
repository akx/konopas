    What is this?

    This guide is an instance of KonOpas, an open-source project I started to provide conventions with easy-to-use mobile-friendly guides. My name is Eemeli Aro, and you can reach me at eemeli@gmail.com.

    This particular instance is a demo version based on KonOpas 0.4.1, using data from LoneStarCon 3, the 2013 World Science Fiction Convention. It has HTML5 caching enabled, so once loaded it should work even when accessed without a working network connection.
